# Run aNd Monitor: An embedded Resource Monitor for Python
<sub><sup>also known as RNM</sup></sub>

## Supported Devices
* CPU
    * Per Core Usage
    * Power / Temperature
* GPU
    * Per GPU Util
    * Per GPU VRAM Usage
    * Power / Temperature
* RAM/Swap
* Hard Drive
    * Bandwidth
    * Total Bytes Read/Written
* Network
    * Bandwidth
    * Total Bytes Received/Sent
